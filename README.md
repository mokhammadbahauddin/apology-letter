# Apology Letter 💌

Sebuah aplikasi web permohonan maaf interaktif yang dibuat dengan HTML, CSS, dan Web Audio API.

## Fitur:
* **Pixel Art Kucing Interaktif:** Animasi kucing berkedip, menangis, menari, dan bisa diusap (petting) dengan efek suara mendengkur.
* **Tombol Menghindar:** Tombol "Ya" yang otomatis berpindah tempat jika didekati untuk membatasi pilihan pacar agar hanya bisa memencet "Tidak".
* **Audio Kotak Musik Chiptune:** Melodi instrumen retro game ceria (Opsi B - Retro Fanfare) yang langsung menyala ketika pacar memencet tombol "Tidak".
* **Layout Responsif Mobile-First:** Tampilan glassmorphism modern yang aman dari notch atau batas layar HP.
